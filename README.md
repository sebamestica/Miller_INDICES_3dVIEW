# Miller Indices 3D Viewer - Professional Crystallography Tool

A high-fidelity, industrial-grade 3D viewer for Miller Indices $(\text{hkl})$ and crystallographic vectors $[\text{hkl}]$. Designed for education and technical analysis in material science and engineering.

![Project Overview](https://img.shields.io/badge/Tech-Three.js-blue) ![Project Status](https://img.shields.io/badge/Status-Complete-green)

## Key features
- **Mathematical Precision**: Accurate $(\text{hkl})$ plane-cube clipping logic for cubic simple systems.
- **Bi-directional Visualization**: Simultaneously visualizes the crystallographic plane and its corresponding normal vector.
- **Mobile-First UX**: Completely responsive, scientific-grade interface that works seamlessly on smartphones and desktops.
- **High-Contrast Design**: Optimized visibility for white-background technical documentation.
- **Interactive Controls**: Full orbit, pan, and zoom functionality for detailed geometric inspection.

## Technical Details
- **Architecture**: Monolithic, single-file HTML/CSS/JS for maximum portability.
- **Engine**: Three.js (v0.160+).
- **Styling**: Vanilla CSS with modern responsive flex/grid layout.

## How to use
1. Open `index.html` in any modern web browser.
2. Input the Miller Indices $(h, k, l)$ in the side/bottom panel.
3. Click "Actualizar Geometría" to see the results.
4. Interact with the 3D viewport using mouse or touch gestures.
